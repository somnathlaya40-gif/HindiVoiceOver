## 📱 How to Use

### Installation

1. Download APK from Releases
2. Allow unknown sources if needed
3. Install the app
4. Grant permissions when prompted

### Using the App

1. **Paste Your Script** - Enter Hindi text you want to convert
2. **Choose Voice** - Select from 12 professional voices
3. **Adjust Speed** - Slow to Fast (5 levels)
4. **Fine-tune Pitch** - Customize the tone
5. **Generate** - Click "GENERATE VOICE"
6. **Listen** - Preview the audio
7. **Download** - Save as MP3

## 🆘 Troubleshooting

**Problem:** Hindi not supported
- **Solution:** Update your phone's language packs

**Problem:** App crashes
- **Solution:** Restart phone, clear cache

**Problem:** Audio quality is poor
- **Solution:** Ensure stable internet connection

**Problem:** Download not working
- **Solution:** Give storage permissions in settings
