## 🔧 Troubleshooting:

**Problem:** App crashes on startup  
**Solution:** Clear app cache (Settings → Apps → Clear Cache)

**Problem:** No sound output  
**Solution:** Check Volume settings + Speaker enabled

**Problem:** Hindi not recognized  
**Solution:** Download Hindi language pack (Settings → Languages)

**Problem:** File not downloading  
**Solution:** Grant Storage permission (Settings → Permissions)
